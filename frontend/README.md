# Personal Portfolio Website in React

This is a personal portfolio website built using React. It showcases projects, skills, and contact information.

## Built Using

- **Front-end library**: React
- **CSS framework**: React-bootstrap
- **CSS animations library**: Animate.css
- **3D Graphics**: Three.js
- **Carousel**: react-multi-carousel

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm test`

Launches the test runner in the interactive watch mode.\

## Features

- **Home Section**: Introduction and tagline with a 3D background.
- **Skills Section**: Displays skills with a carousel.
- **Projects Section**: Showcases projects with images and descriptions.
- **Contact Section**: Contact form to send messages.
